Prefer the command line?



Make sure you have pip, the Python package manager installed. Needs to be Python 3, not Python 2. Lastly, make sure you have a Tor SOCKS proxy running locally (127.0.0.1:9050)


pip3 install doublemixer


doublemixer mix --currency bitcoin --output_address (your address)


DoubleMixer Python source code is also available on Github.






DoubleMixer is available over clearnet or Tor Hidden Services (.onion).
